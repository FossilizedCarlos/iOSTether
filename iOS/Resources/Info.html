This app is a SOCKS Proxy Server. It allows you to connect your laptop to the Internet through the 3G/Edge connection of your iPhone or iPad. In order for this to work you need to follow few steps (instructions for Mac):
  • Step 1: Mac: Start an ad-hoc Wi-Fi network:
    • System Preferences → Network → AirPort
    • Click on Network Name and select Create Network
    • Choose a name for your network (e.g. mywifi) and confirm it with OK and Apply
  • Step 2: iPhone: Connect your iPhone to the ad-hoc wifi network:
    • Settings → Wi-Fi
    • Select your ad-hoc network
  • Step 3: iPhone: Start the SOCKS Proxy Server:
    • Run this Tethering app on your iPhone
    • Press Start
  • Step 4: Mac: Configure your Mac to use SOCKS:
    • System Preferences → Network → Advanced... → Proxies → SOCKS Proxy
    • In the SOCKS Proxy Server field enter the address and port that appear on your iPhone screen
    • Press OK and Apply
